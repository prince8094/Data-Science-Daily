## 1. Theoretical Concepts: Functional Tools & OOP Magic

### Functional Programming Tools
These functions allow for concise data manipulation, often used with **Lambda** (anonymous) functions.
* **Lambda Function:** A small, one-line function without a name.
  * Syntax: `lambda arguments : expression`
* **Map Function:** Applies a given function to every item in an iterable (list, tuple, etc.) and returns a map object.
* **Filter Function:** Filters an iterable based on a condition; returns only the items that evaluate to `True`.
* **Reduce Function:** (Imported from `functools`) Reduces an iterable to a single cumulative value by applying a function repeatedly.
* **Zip Function:** Combines multiple iterables into a single iterator of tuples.


### Magic (Dunder) Methods in OOP
Magic methods start and end with double underscores (`__`). They allow you to define how objects behave with standard Python operators.
* **Comparison:** `__eq__` (==), `__lt__` (<), `__gt__` (>).
* **Arithmetic:** `__add__` (+), `__sub__` (-), `__mul__` (*), `__truediv__` (/).
* **Unary & Assignment:** `__neg__` (- operator), `__iadd__` (+=).
* **Container/Sequence:**
    * `__len__`: Returns length of the object.
    * `__getitem__`: Accesses an item via index/key.
    * `__setitem__`: Sets a value for a key.
    * `__contains__`: Handles the `in` operator.

### Character Encoding (ASCII & Unicode)
Python provides built-in functions to handle the mapping between characters and their integer positions.
* **`ord(char)`:** Converts a string character to its integer ASCII/Unicode value.
* **`chr(int)`:** Converts an integer value back to its string character.
* **ASCII Ranges:** * `0-9`: 48–57
    * `A-Z`: 65–90
    * `a-z`: 97–122
* **Representation:**
    * `repr()`: Returns a string representation of an object (shows unicode characters as they are).
    * `ascii()`: Returns an ASCII-safe string, escaping non-ASCII characters using Unicode sequences.

## 2. Full Python Implementations

```python
from functools import reduce

# --- 1. Functional Tools ---
nums = [1, 2, 3, 4, 2, 1, 2]

# Lambda & Map: Square each number
squared = list(map(lambda x: x**2, nums))
print(f"Squared: {squared}")

# Filter: Only even numbers
evens = list(filter(lambda x: x % 2 == 0, nums))
print(f"Evens: {evens}")

# Reduce: Sum of all numbers
total_sum = reduce(lambda x, y: x + y, nums)
print(f"Total Sum: {total_sum}")

# Zip: Pair names and ages
names = ['Ashish', 'Ankur', 'Avinash']
ages = [25, 27, 22]
paired = list(zip(names, ages))
print(f"Paired Data: {paired}")

# --- 2. Magic Methods Example ---
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
        
    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)
    
    def __repr__(self):
        return f"Vector({self.x}, {self.y})"

v1 = Vector(2, 3)
v2 = Vector(4, 5)
print(f"Vector Addition: {v1 + v2}")

# --- 3. ASCII & Encoding ---
char_a = 'A'
code_a = ord(char_a)
print(f"ASCII of {char_a}: {code_a}")
print(f"Character for 97: {chr(97)}")

# repr vs ascii
special_char = 'ñ'
print(f"repr: {repr(special_char)}")
print(f"ascii: {ascii(special_char)}")

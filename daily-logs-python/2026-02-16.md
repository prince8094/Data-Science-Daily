## 1. Theoretical Concepts: Modules & Inbuilt Libraries

### Working with Modules
Modules allow you to organize code into separate files and reuse them.
* **Importing:** * `import filename`: Imports the entire module.
    * `from filename import functionname`: Imports only a specific function.
    * `from filename import *`: Imports everything from the module.

### Inbuilt Libraries (Standard Library)
Python comes with several built-in modules to handle math, data structures, and time.

#### A. `math` Module
Used for mathematical constants and functions:
* **Functions:** `ceil(x)`, `floor(x)`, `trunc(x)`, `exp(x)`, `log10(x)`, `sin(x)`, `factorial(x)`.
* **Constants:** `math.pi` ($\pi$), `math.e` ($e$).

#### B. `random` Module
Used for generating random numbers and making random selections:
* `seed(x)`: Initializes the random number generator (makes results reproducible).
* `randint(start, stop)`: Returns a random integer between the range.
* `choice(list)`: Picks one random element from a list.
* `sample(list, k)`: Picks `k` unique random elements.
* `uniform(a, b)`: Returns a random float between $a$ and $b$.

#### C. `datetime` Module
Used for manipulating dates and times:
* `datetime.now()`: Current date and time.
* `strftime("format")`: Converts a datetime object into a readable string (e.g., `"%d/%m/%Y"`).
* **Date Arithmetic:** Subtracting two dates gives a `timedelta` object.

#### D. `collections` Module
Provides advanced data structures:
* **`Counter`:** A dictionary subclass for counting hashable objects.
* **`defaultdict`:** A dictionary that provides a default value for keys that don't exist yet.
* **`OrderedDict`:** A dictionary that remembers the order in which items were inserted.

#### E. `string` Module
Common string constants:
* `ascii_letters`, `ascii_lowercase`, `digits`, `hexdigits`.
* `punctuation`: A string containing all punctuation characters.

## 2. Full Python Implementations

```python
import math
import random
from datetime import datetime
from collections import Counter, defaultdict, OrderedDict
import string

# --- 1. Math Operations ---
print(f"Pi: {math.pi}")
print(f"Factorial of 5: {math.factorial(5)}")
print(f"Log10 of 100: {math.log10(100)}")

# --- 2. Random Selections ---
random.seed(40)
my_list = [1, 2, 3, 4, 5]
print(f"Random Choice: {random.choice(my_list)}")
print(f"Random Int (1-10): {random.randint(1, 11)}")

# --- 3. Datetime Formatting ---
now = datetime.now()
formatted_date = now.strftime("%d/%m/%Y %H:%M:%S")
print(f"Current Date: {formatted_date}")

# --- 4. Collections Module ---
# Counter: List to frequency dict
data = [1, 2, 3, 4, 5, 1, 2, 1]
print(f"Frequency Count: {Counter(data)}")

# Defaultdict: Prevents KeyErrors
d = defaultdict(int)
d['a'] += 1
print(f"DefaultDict Value: {d['a']}")

# --- 5. String Constants ---
print(f"All Punctuations: {string.punctuation}")
# Remove punctuation example
sample_str = "Hello, World!"
clean_str = "".join([c for c in sample_str if c not in string.punctuation])
print(f"Cleaned String: {clean_str}")

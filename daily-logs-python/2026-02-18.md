## 1. Theoretical Concepts: Errors & Exception Handling

### Common Types of Errors
1.  **Syntax Error:** Code does not follow Python's rules (e.g., missing colons).
2.  **Runtime Error:** Occurs during execution (e.g., division by zero).
3.  **Logical Error:** Code runs but gives the wrong output.
4.  **Name Error:** Using a variable that hasn't been defined.
5.  **Type Error:** Operation performed on an inappropriate data type.
6.  **Index Error:** Accessing an out-of-range index in a list.
7.  **Key Error:** Accessing a non-existent key in a dictionary.
8.  **Attribute Error:** Attempting to access an attribute/method a variable doesn't have.
9.  **Indentation Error:** Incorrect spacing/tabs.
10. **Import Error:** Module or function cannot be found.
11. **Value Error:** Function receives an argument of correct type but improper value.

### Exception Handling (`try`, `except`, `else`, `finally`)
* **`try`:** Block where you write code that *might* throw an error.
* **`except`:** Block that executes if an error occurs. You can catch specific errors (like `ZeroDivisionError`).
* **`else`:** Executes only if **no error** occurred in the try block.
* **`finally`:** Executes **every time**, regardless of whether an error occurred or not. Often used for cleaning up resources (like closing files).

### File Handling
Used to read from or write to external text files.
* **Modes:** `'r'` (Read), `'w'` (Write - overwrites), `'a'` (Append).
* **Basic Syntax:**
    * `fd = open('filename', 'mode')`
    * `content = fd.read()`
    * `fd.close()` (Always close the file to free up memory).

### Text Processing Logic
* **Split by Paragraphs:** `txt.split("\n\n")` (returns a list of paragraphs).
* **Split by Sentences:** `txt.split(".")`.
* **Split by Words:** `txt.split(" ")`.
* **Pattern Removal:** Using a loop and `.replace()` to remove specific bracketed patterns (e.g., `[7]`, `[8]`).

## 2. Full Python Implementations

```python
# --- 1. Basic Exception Handling ---
try:
    x = 10
    print(x / 0)
except ZeroDivisionError:
    print("Error: Cannot divide by zero! Enter a number other than 0.")
except Exception as e:
    print(f"An unknown error occurred: {e}")
else:
    print("Division successful!")
finally:
    print("Execution complete (this always runs).")

# --- 2. File Handling & Processing ---
file_path = 'sample_notes.txt'

# Writing to a file
with open(file_path, 'w') as f:
    f.write("This is a sample paragraph.\n\nIt has [7] reference marks [8].")

# Reading and Processing
try:
    with open(file_path, 'r') as fd:
        txt = fd.read()
        print(f"File Content:\n{txt}")
        
        # Text Analysis
        paragraphs = txt.split("\n\n")
        words = txt.split(" ")
        print(f"No. of Paragraphs: {len(paragraphs)}")
        print(f"No. of Words: {len(words)}")

        # Pattern Cleaning (Removing [7], [8]... [18])
        for i in range(7, 19):
            pattern = "[" + str(i) + "]"
            txt = txt.replace(pattern, "")
        
        print(f"Cleaned Text: {txt}")

except FileNotFoundError:
    print("The specified file was not found.")

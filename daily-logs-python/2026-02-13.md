## 1. Core Concepts: Dictionary Comprehension & Sets

### Dictionary Comprehension
Similar to list comprehension, this is a concise way to create dictionaries from iterables or by transforming existing dictionaries.
* **Basic Syntax:** `{key: value for item in iterable}`
* **Creating from two lists:** Using `zip()` to pair keys and values: `{k: v for (k, v) in zip(keys, values)}`
* **Special Keys:** You can modify keys during creation (e.g., converting integers to strings: `str(i)`).

### Supporting Data Structures: Sets vs. Tuples

| Feature | Sets `{ }` | Tuples `( )` |
| :--- | :--- | :--- |
| **Mutability** | Mutable (can add/remove) | **Immutable** (cannot change) |
| **Uniqueness** | **No Duplicates** (Unique values only) | Allows Duplicates |
| **Ordering** | Unordered | Ordered |
| **Underlying** | Known as a **Hash Table** | Linear Sequence |

### Sets in Depth
Sets are perfect for membership testing and removing duplicates from a collection.
* **Operations:**
    * `.add(x)`: Adds an element.
    * `.pop()`: Removes an arbitrary element.
    * `.discard(x)`: Removes a specific element (does not throw error if missing).
* **Frozenset:** An immutable version of a set.
* **Mathematical Set Operations:**
    * **Union (`|`):** Combines all unique elements from both sets.
    * **Intersection (`&`):** Only elements present in both sets.
    * **Difference (`-`):** Elements in Set 1 but not in Set 2.

## 2. Python Code Implementation


```python
# --- 1. Dictionary Comprehension ---
# Squares of numbers 1-5
sq_dict = {i: i**2 for i in range(1, 6)}
print(f"Squares Dict: {sq_dict}")

# List to Dict (Length of strings)
fruits = ['Apple', 'Banana', 'Grapes']
fruit_len = {item: len(item) for item in fruits}

# Creating from two lists
keys = ['a', 'b', 'c', 'd']
vals = [1, 2, 3, 4]
combined_dict = {k: v for (k, v) in zip(keys, vals)}

# --- 2. Sets: Creation & Iteration ---
my_set = {1, 2, 3, 4}
my_set.add(6)
my_set.discard(3) # Removes 3

print(f"Is 2 in set? {2 in my_set}")

# --- 3. Mathematical Set Operations ---
set1 = {1, 2, 3, 4, 5}
set2 = {4, 5, 6, 7, 8}

print(f"Union: {set1 | set2}")        # {1, 2, 3, 4, 5, 6, 7, 8}
print(f"Intersection: {set1 & set2}") # {4, 5}
print(f"Difference: {set1 - set2}")   # {1, 2, 3}

# --- 4. Nested (Multidimensional) Dictionaries ---
# Accessing deeper levels
dct = {
    1: {'name': 'Ashish', 'phone': 12345},
    2: {'name': 'Prince', 'phone': 22345}
}

# Iterating through keys to access nested values
for i in dct.keys():
    print(f"ID: {i}, Name: {dct[i]['name']}, Phone: {dct[i]['phone']}")

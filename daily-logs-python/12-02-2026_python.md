## 1. Core Concepts: Advanced Lists & Dictionaries

### List Comprehension
List comprehension provides a concise and powerful way to create lists. 
* **Benefits:** Cleaner code, more readable, and faster execution.
* **Syntax:** `[expression for item in iterable if condition]`
* **Nested Comprehension:** Used for creating or flattening multidimensional lists (e.g., creating a 5x3 matrix).


### Dictionaries (`dict`)
Dictionaries store data in **Key:Value** pairs. They are unordered (in older versions), mutable, and indexed by keys.
* **Keys:** Must be **unique** and of an immutable data type (int, float, string). If a key is duplicated, the last value assigned is remembered.
* **Accessing:** Use `dct[key]` or the safer `dct.get(key)` method.
* **Operations:**
    * **Adding/Updating:** `dct[key] = value`
    * **Deleting:** `del dct[key]` or `dct.pop(key)`
    * **Merging:** `dct1.update(dct2)` updates `dct1` with pairs from `dct2`.
* **Checking Presence:** Use the `in` keyword (e.g., `1 in dct` returns `True` or `False`).

### Iterating through Dictionaries
1. **Keys only:** `for i in dct.keys():`
2. **Values only:** `for i in dct.values():`
3. **Both (Items):** `for k, v in dct.items():` (Returns pairs as tuples).

### Multidimensional (Nested) Dictionaries
Dictionaries can contain other dictionaries as values. This is useful for representing structured data like a database of users.
* **Accessing:** `dct[outer_key][inner_key]`
* **DefaultDict:** A specialized dictionary from the `collections` module that provides a default value for a nonexistent key.

## 2. Python Code Implementation

```python
# --- 1. List Comprehension ---
# Regular way vs Comprehension
numbers = [1, 2, 3, 4, 5, 6]
squares = [i**2 for i in numbers] # [1, 4, 9, 16, 25, 36]

# Filtering even numbers from range(20)
evens = [i for i in range(20) if i % 2 == 0]
print(f"Evens: {evens}")

# --- 2. Multidimensional List (Matrix) ---
# Creating a 5x3 matrix using comprehension
matrix = [[j for j in range(3)] for i in range(5)]
# Flattening the matrix
flattened = [num for row in matrix for num in row]
print(f"Flattened Matrix: {flattened}")

# --- 3. Dictionary Basics ---
user_dct = {1: 'Ashish', 2: 'Prince', 3: 'Riya'}

# Accessing & Updating
print(user_dct.get(2)) # Output: 'Prince'
user_dct[6] = 'Sadaf'  # Adding new
user_dct[5] = 'Prince' # Updating existing

# --- 4. Iterating through Dictionary ---
for k, v in user_dct.items():
    print(f"Key: {k}, Value: {v}")

# --- 5. Nested Dictionaries ---
students = {
    1: {'name': 'Ashish', 'phone': 12345},
    2: {'name': 'Prince', 'phone': 22345}
}

# Accessing nested data
print(f"Student 1 Name: {students[1]['name']}")

# Iterating through nested dict
for sid, info in students.items():
    print(f"ID: {sid}, Name: {info['name']}, Phone: {info['phone']}")

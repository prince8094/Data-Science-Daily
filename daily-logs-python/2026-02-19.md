## 1. Theoretical Concepts: Advanced File Writing & OS Library

### Writing Text on Text Files
* **Creation:** Using `open("filename", 'w')` creates a new file. If the file already exists, 'w' mode will overwrite it.
* **Writing:** `fd.write("text")` writes a single string to the file. 
* **Append Mode ('a'):** When performing write operations in append mode, Python adds the new text to the end of the existing content instead of overwriting it.
* **Multi-line Writing:** To write on a new line, you must explicitly add the newline character `\n` at the end of your string.

### Inventory Management System (Project Logic)
A basic system to manage stock requires tracking four primary attributes for each item:
1. **Product ID:** Unique identifier for the item.
2. **Name of Product:** Descriptive title.
3. **Price of Product:** Cost per unit.
4. **Quantity:** Current stock level.

### System Commands & OS Library
The `os` library in Python allows you to execute operating system tasks directly from your code.

#### Comparison of Terminal Commands
| Task | Windows (CMD) | Linux/Mac (Terminal) |
| :--- | :--- | :--- |
| List Files | `dir` | `ls` |
| Clear Screen | `cls` | `clear` |
| Make Directory | `mkdir` | `mkdir` |
| Remove Directory | `rmdir` | `rmdir` |
| Change Directory | `cd` | `cd` |
| Delete File | `del` / `erase` | `rm` |
| Move/Rename | `move` / `rename` | `mv` |

#### Python `os` Module Functions
* **`os.getcwd()`**: Returns the **Current Working Directory**.
* **`os.listdir()`**: Returns a list of all files and folders in the current directory.
* **`os.chdir("foldername")`**: Changes the current working directory.
* **`os.chdir("../")`**: Moves one level back in the directory tree.
* **`os.mkdir("name")`**: Creates a single new folder.
* **`os.makedirs("path/to/folder")`**: Creates nested directories (folders inside folders) in one go.

## 2. Full Python Implementations


```python
import os

# --- 1. Advanced File Writing (Append Mode) ---
file_name = "Data.txt"

# Creating and writing the first line
with open(file_name, 'w') as fd:
    fd.write("I am Prince\n")

# Appending multiple lines
with open(file_name, 'a') as fd:
    fd.write("Line 2: Learning Python File Handling\n")
    fd.write("Line 3: Adding a third line using append mode\n")

# --- 2. OS Library Operations ---
# Get current location
current_dir = os.getcwd()
print(f"Current Directory: {current_dir}")

# List all files
print(f"Files in directory: {os.listdir()}")

# Create a nested directory structure
folder_path = "Project/Data/Logs"
if not os.path.exists(folder_path):
    os.makedirs(folder_path)
    print(f"Created nested folders: {folder_path}")

# Move back one directory
os.chdir("../")
print(f"New Directory: {os.getcwd()}")

# --- 3. Inventory System Concept (Dict Representation) ---
inventory = {
    "101": {"name": "Laptop", "price": 50000, "quantity": 10},
    "102": {"name": "Mouse", "price": 500, "quantity": 50}
}

# Update quantity for Product ID 101
inventory["101"]["quantity"] -= 1
print(f"Updated Inventory: {inventory['101']}")

## 1. Theoretical Concepts & Intuition

### Tuples (`tpl`)
* **Definition:** Ordered, **immutable** sequence. Elements cannot be changed after definition.
* **Key Operations:**
    * `tpl[0]`: **Accessing** via indexing.
    * `tpl[1:3]`: **Slicing** for a sub-range.
    * `tpl1 + tpl2`: **Concatenation** to create a new tuple.
    * `tpl.count(x)`: Frequency of an element.
    * `tpl.index(x)`: First index position of an element.
* **Unpacking with Asterisk (`*`):** * `a, *b, c = tup`
  * The `*` captures multiple middle elements into a list. For `tup = (1, 2, 3, 4, 5)`, `a=1`, `b=[2, 3, 4]`, and `c=5`.

### Set Operations (`s1`, `s2`)
Sets are unordered collections of unique items supporting relational logic:
* `s1 == s2`: S1 is equivalent to S2.
* `s1 <= s2`: S1 is a **subset** of S2.
* `s1 < s2`: S1 is a **proper subset** of S2.
* `s1 >= s2`: S1 is a **superset** of S2.
* `s1 | s2`: **Union** (All unique elements).
* `s1 & s2`: **Intersection** (Common elements only).
* `s1 - s2`: **Difference** (Elements in S1 but not in S2).
* `s1 ^ s2`: **Symmetric Difference** (All elements minus the intersection).

### Functions
* **Purpose:** Code Reusability and Modularity.
* **Default Parameters:** Assigning a default value (e.g., `def greet(name='---')`). If no argument is passed, the default is used.
* **Return:** Sends a result back to the caller.

### Object-Oriented Programming (OOP) Basics
* **Class & Object:** Class is the blueprint; Object is the instance (e.g., `p1 = Person()`).
* **`self`:** Reference to the current instance; used to access class variables.
* **`__init__` (Constructor):** Executes **automatically** upon object creation to initialize attributes like `name` and `age`.

## 2. Full Python Implementations

```python
# --- TUPLE & UNPACKING ---
tup = (1, 2, 3, 4, 5)
a, *b, c = tup
print(f"a: {a}, b: {b}, c: {c}") # b becomes [2, 3, 4]

# --- SET OPERATIONS ---
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}
print(f"Union: {s1 | s2}") 
print(f"Symmetric Diff: {s1 ^ s2}")

# --- FUNCTIONS WITH DEFAULTS ---
def greet(name='---'):
    print(f"Hello {name}")

greet("Ashish") 
greet() # Uses default '---'

# --- OOP: CLASS & INIT ---
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def display(self):
        print(f"Name: {self.name}, Age: {self.age}")

# Create Instance
p1 = Person("Prince", 20)
p1.display()

class Math:
    name = "Ashish"
    def greet(self):
        print(f"Hello {self.name}")

m1 = Math()
m1.greet()

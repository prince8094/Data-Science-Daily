## 1. Theoretical Concepts & Intuition

**Linear Regression** is a supervised learning algorithm that fits a straight line to observed data to demonstrate a linear relationship between variables and make predictions.

* **Supervised Learning:** Regression (predicting continuous outcomes).
* **Unsupervised Learning:** Clustering and anomaly detection.
* **Reinforcement Learning:** Supervised learning using simulations.

### Benefits & Downsides
* **Benefits:** 1. Allows predictions on unseen data.
    2. Helps analyze and hypothesize causal relationships between correlated variables.
* **Downsides:** 1. We cannot expect every outcome to fall exactly on the line.
    2. Avoid using linear regression for predictions outside the range of your training data (Extrapolation).

## 4. Full Python Implementation (Scikit-Learn)

```python
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# Initialize Data
data = {
    'Hours': [1, 2, 3, 4, 5, 6, 7, 8],
    'Score': [10, 20, 30, 40, 50, 60, 70, 80]
}

df = pd.DataFrame(data)
print("Dataset:\n", df)

# Prepare Variables
X = df[['Hours']] # Independent
y = df['Score']   # Dependent

# Scikit-Learn Model
model = LinearRegression()
model.fit(X, y)

print(f"\nIntercept (b): {model.intercept_}")
print(f"Coefficient (m): {model.coef_[0]}")

# Plotting the results
plt.scatter(X, y, color='blue', label='Actual Points')
plt.plot(X, model.predict(X), color='red', label='Linear Fit')
plt.xlabel('Hours')
plt.ylabel('Score')
plt.legend()
plt.show()

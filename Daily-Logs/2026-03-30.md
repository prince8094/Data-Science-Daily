## 1. Theoretical Concepts: Advanced Distributions & Testing

### Specialized Distributions
Beyond the core distributions, several others are critical for specific modeling tasks:
* **Beta Distribution:** A continuous distribution defined on the interval $[0, 1]$. Often used to model probabilities or proportions.
* **Gamma Distribution:** Used to model the waiting time until a specific number of events occur.
* **Weibull Distribution:** A specialized version of the exponential distribution used for reliability and life-data analysis.
* **Geometric Distribution:** Models the number of failures required until the first success occurs in a series of Bernoulli trials.

### Central Limit Theorem (CLT)
The CLT is the foundation of inferential statistics. It states that if you take sufficiently large samples ($n \ge 30$) from any population with mean $\mu$ and standard deviation $\sigma$, the distribution of the **sample means** will be approximately normal.
* **Sample Mean:** $\text{mean} = \mu$
* **Standard Error (Std):** $\text{std} = \sigma / \sqrt{n}$


### Hypothesis Testing Framework
A systematic way to test a claim or objective by deciding to either **Reject** or **Fail to Reject** the Null Hypothesis.
* **Null Hypothesis ($H_0$):** Represents "no effect" or "no relationship."
* **Alternative Hypothesis ($H_1$):** Represents the effect or relationship you are trying to prove.
* **Test Types:**
    * **Parametric Tests:** Assume a specific distribution (e.g., Normal) and independence.
    * **Non-Parametric Tests:** Do not assume a specific distribution (useful for categorical or non-normal data).

### Steps for Hypothesis Testing
1. **Define $H_0$ and $H_1$.**
2. **Identify the test statistic** to be used (Z-test, T-test, etc.).
3. **Decide the Significance Level ($\alpha$):** Usually 0.05 (5%).
4. **Calculate the p-value:** The probability of observing the result if $H_0$ is true.
5. **Make a decision:** If $p\text{-value} < \alpha$, Reject $H_0$.

## 2. Python Implementation & Z-Test

### The Z-Test
Used for testing the population mean when the sample size is large ($n > 30$) and the population variance is known.
* **Formula:** $$Z = \frac{\bar{x} - \mu}{\sigma / \sqrt{n}}$$
  * $\mu$ = Population Mean
  * $\sigma$ = Population Std
  * $\bar{x}$ = Sample Mean
  * $n$ = Sample Size

```python
import numpy as np
from scipy import stats

# --- 1. Central Limit Theorem Demonstration ---
# Population (Non-normal)
population = np.random.exponential(scale=2, size=10000)
# Sampling distribution of the mean
sample_means = [np.mean(np.random.choice(population, size=50)) for _ in range(1000)]

# --- 2. Z-Test Implementation ---
def z_test(sample_mean, pop_mean, pop_std, n):
    z_score = (sample_mean - pop_mean) / (pop_std / np.sqrt(n))
    # Two-tailed p-value
    p_value = 2 * (1 - stats.norm.cdf(abs(z_score)))
    return z_score, p_value

# Parameters
mu, sigma = 100, 15  # Population
x_bar, n = 105, 40   # Sample
alpha = 0.05

z, p = z_test(x_bar, mu, sigma, n)

print(f"Z-Statistic: {z:.4f}")
print(f"P-Value: {p:.4f}")

if p < alpha:
    print("Decision: Reject Null Hypothesis (Statistically Significant)")
else:
    print("Decision: Fail to Reject Null Hypothesis")

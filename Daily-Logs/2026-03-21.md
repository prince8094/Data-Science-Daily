## 1. Core Concepts: Systems of Equations & Eigendecomposition

### Solving Systems of Equations
To solve a system of linear equations (like the 3-variable system in the notes), we represent it in matrix form:
$$AX = B$$
Where:
* **$A$**: The coefficient matrix.
* **$X$**: The column vector of variables ($x, y, z$).
* **$B$**: The column vector of constants.

To find $X$, we "undo" matrix $A$ by multiplying both sides by its inverse ($A^{-1}$):
$$A^{-1}AX = A^{-1}B \implies IX = A^{-1}B \implies X = A^{-1}B$$

### Eigenvectors & Eigenvalues
* **Matrix Decomposition:** The process of breaking a matrix into basic components (similar to factoring numbers).
* **Eigendecomposition:** A specific type of decomposition used in Machine Learning and **Principal Component Analysis (PCA)**. It only works on **square matrices**.
* **Eigenvalues ($\lambda$):** Scalars that represent the factor by which the eigenvector is stretched during a transformation.
* **Eigenvectors ($v$):** Vectors that do not change direction during a linear transformation.
* **The Equation:** $$A \cdot v = \lambda \cdot v$$


### Rebuilding Matrix A
You can rebuild the original matrix $A$ using its eigenvalues and eigenvectors:
$$A = V \Lambda V^{-1}$$
* $V$ = Matrix of Eigenvectors.
* $\Lambda$ = Diagonal matrix of Eigenvalues.
* $V^{-1}$ = Inverse of the Eigenvector matrix.

## 2. Python Implementation (SymPy & NumPy)

### A. Solving Equations using SymPy
```python
from sympy import Matrix

# Define Coefficient Matrix A and Constant Matrix B
A = Matrix([
    [4, 2, 4],
    [5, 3, 7],
    [9, 3, 6]
])
B = Matrix([44, 56, 72])

# Find Inverse of A
inverse_A = A.inv()

# Solve for X (X = A^-1 * B)
X = inverse_A * B
print(f"Solution Matrix X: {X}") # Output: Matrix([[2], [34], [-8]])

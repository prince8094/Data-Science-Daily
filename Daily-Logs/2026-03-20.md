## 1. Core Concepts: Linear Algebra

### Vectors
* **Definition:** A vector is a quantity having both magnitude and direction, represented as:
  $$\vec{v} = \begin{bmatrix} x \\ y \end{bmatrix}$$
* **Vector Addition:** Geometrically follows the triangle law. In Python, we use `numpy` for element-wise addition.
* **Scaling Vectors:** Multiplying a vector by a scalar stretches or squishes it (e.g., $2\vec{v}$).



[Image of vector addition triangle law]


### Span & Linear Independence
* **Span:** The set of all possible linear combinations of a set of vectors. 
* **Linear Independent:** Two vectors having different directions (not on the same line) have an **Infinite Span**, meaning they can reach any point in the 2D plane.
* **Linear Dependent:** Vectors on the same line. Their span is limited to just that line.

### Linear Transformations
Transformations involve stretching, squishing, shearing, or rotating the basis vectors ($\hat{i}$ and $\hat{j}$).
* Standard basis: $\hat{i} = \begin{bmatrix} 1 \\ 0 \end{bmatrix}$, $\hat{j} = \begin{bmatrix} 0 \\ 1 \end{bmatrix}$.
* If we transform them to $3\hat{i}$ and $2\hat{j}$, any vector $\vec{v}$ in that space is transformed accordingly.



### Matrix Operations
* **Matrix Multiplication:** Acts as a linear transformation on a vector.
  $$\begin{bmatrix} a & b \\ c & d \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} ax + by \\ cx + dy \end{bmatrix}$$
* **Determinants:** Represents the factor by which areas are scaled during a transformation. If the determinant is 0, the space is squished into a lower dimension (like a line or a point).
* **Transpose:** Swapping rows with columns ($Rows \leftrightarrow Columns$).

### Types of Matrices
1. **Square Matrix:** Equal rows and columns ($n \times n$).
2. **Identity Matrix ($I$):** Square matrix with 1s on the diagonal and 0s elsewhere.
3. **Inverse Matrix ($A^{-1}$):** Undoes the transformation of matrix $A$. $AA^{-1} = I$.
4. **Diagonal Matrix:** All elements are 0 except the main diagonal.
5. **Sparse Matrix:** Contains mostly zeros. Important for computational efficiency in large datasets.

## 2. Python Implementation (using NumPy)

```python
import numpy as np

# --- 1. Vector Operations ---
v = np.array([3, 2])
w = np.array([2, -1])

# Addition
vec_sum = v + w 
print(f"Vector Sum: {vec_sum}")

# --- 2. Linear Transformation (Matrix-Vector Mult) ---
# Transformation matrix
basis = np.array([
    [3, 0],
    [0, 2]
])
v_orig = np.array([1, 1])

# Using .dot() for matrix multiplication
new_v = basis.dot(v_orig)
print(f"Transformed Vector: {new_v}") # Output: [3, 2]

# --- 3. Determinants & Transpose ---
matrix_a = np.array([[1, 2], [3, 4]])

# Transpose
print(f"Transpose:\n{matrix_a.T}")

# Determinant
det = np.linalg.det(matrix_a)
print(f"Determinant: {det:.2f}")

# --- 4. Special Matrices ---
identity = np.eye(3) 
print(f"3x3 Identity Matrix:\n{identity}")

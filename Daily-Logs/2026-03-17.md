### 📖 Theoretical Concepts

#### 1. Calculus for Machine Learning
* **Linear Functions:** $f(x, y) = 2x + 3y$ represents a plane in 3D space.
* **Limits:** Using `SymPy` to find the value a function approaches as $x$ reaches a certain point (e.g., $\infty$).
* **Derivatives:** The slope or rate of change of a function. 
    * If $f(x) = x^2$, the derivative is $f'(x) = 2x$.
* **Partial Derivatives:** Calculating the derivative with respect to one variable while holding others constant.
    * $\frac{\partial}{\partial x}(2x^3 + 3y^3) = 6x^2$
* **Integrals:** Finding the area under the curve.
    * $\int_{0}^{1} (x^2 + 1) dx = \frac{4}{3}$

#### 2. Statistics & Distributions
* **Central Tendency:** * **Mean ($\bar{x}$):** $\frac{\sum x_i}{n}$
    * **Median:** The middle value of a sorted dataset.
    * **Mode:** The most frequent value.
* **Skewness:** * **Left Skew:** Mean < Median < Mode
    * **Right Skew:** Mode < Median < Mean
* **Spread:**
    * **Variance ($\sigma^2$):** $\frac{\sum (x_i - \mu)^2}{N}$
    * **Standard Deviation ($\sigma$):** $\sqrt{Variance}$

#### 3. Probability Theory
* **Odds:** $O(x) = \frac{P(x)}{1 - P(x)}$
* **Joint Probability (AND):** $P(A \cap B) = P(A) \times P(B)$ (for independent events).
* **Union Probability (OR):** $P(A \cup B) = P(A) + P(B) - P(A \cap B)$.
* **Bayes' Theorem:** $P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$
* **Beta Distribution:** Used to model the likelihood of different underlying probabilities for an event given $\alpha$ (successes) and $\beta$ (failures).

### 💻 Python Implementation

```python
import sympy
from sympy import symbols, limit, diff, integrate, oo
from scipy.stats import beta
import numpy as np

# Calculus: Derivative & Integral
x = symbols('x')
f = x**2 + 1
print(f"Derivative: {diff(f, x)}")
print(f"Integral (0 to 1): {integrate(f, (x, 0, 1))}")

# Statistics: Mean and Median
data = [1, 2, 3, 4, 5, 6]
print(f"Mean: {np.mean(data)}")
print(f"Median: {np.median(data)}")

# Probability: Beta Distribution
# Likelihood of success < 90% with 8 successes and 2 failures
prob = beta.cdf(0.90, 8, 2)
print(f"Probability of success < 90%: {prob:.4f}")

# 23/03/2026 : Residuals and Optimization Techniques

## 1. Theoretical Concepts & Intuition

To find the "Best-Fit Line," we must minimize the distance between the data points and the line.

### Residuals (Errors)
A **Residual** is the numeric difference between the line and the actual points.
* **Positive Residual:** Points above the line.
* **Negative Residual:** Points below the line.
* **Goal:** Minimize the Sum of Squared Errors:
$$\sum (Y_{actual} - Y_{predicted})^2 \rightarrow \text{minimize}$$

---

## 2. Time/Space Complexity Table

| Approach | Time Complexity | Space Complexity |
| :--- | :--- | :--- |
| **Closed Form** | $O(n)$ | $O(1)$ |
| **Matrix Inversion** | $O(k^3)$ | $O(k^2)$ |
| **Gradient Descent** | $O(i \cdot n \cdot k)$ | $O(k)$ |

---

## 3. Summary of all Approaches

1.  **Closed Form Equation:** Simple math for single input variables.
2.  **Matrix Inversion (Normal Equation):** Linear algebra approach for multiple variables.
3.  **Gradient Descent:** Iterative "hill climbing" optimization.
4.  **Matrix Decomposition:** Numerically stable methods like SVD.

## 4. Full Python Implementation (Matrix & Math)

```python
import numpy as np
import pandas as pd

# Data
data = {'Hours': [1, 2, 3, 4, 5, 6, 7, 8], 'Score': [10, 20, 30, 40, 50, 60, 70, 80]}
df = pd.DataFrame(data)
X = df[['Hours']]
y = df['Score']

# APPROACH 1: Closed Form Equation (m and b)
n = len(df)
sum_x = df['Hours'].sum()
sum_y = df['Score'].sum()
sum_xy = (df['Hours'] * df['Score']).sum()
sum_x2 = (df['Hours']**2).sum()

m = (n*sum_xy - sum_x*sum_y) / (n*sum_x2 - (sum_x)**2)
b = (sum_y / n) - m * (sum_x / n)
print(f"Closed Form -> Slope: {m}, Intercept: {b}")

# APPROACH 2: Matrix Inversion Technique
# Adding a column of 1s for the intercept (b)
X_mat = np.hstack((np.ones((len(X), 1)), X))
y_mat = y.values.reshape(-1, 1)

# Formula: b = inv(X.T @ X) @ (X.T @ y)
beta = np.linalg.inv(X_mat.T @ X_mat) @ (X_mat.T @ y_mat)

print(f"Matrix Inversion -> Intercept: {beta[0][0]}, Slope: {beta[1][0]}")
